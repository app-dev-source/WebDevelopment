# 🃏 Pokémon Card Game

A dynamic, web-based card game featuring a Pokémon theme. This project is built using standard front-end web technologies, offering a smooth and interactive user experience.

## 🚀 Features

* **Interactive Card Deck:** Visual stack of cards with a sleek Pokéball back-design.
* **Smooth Animations:** Engaging hover effects and card-shuffling/dealing transitions.
* **Responsive Design:** Optimized layout that centers and scales beautifully across various screen dimensions.
* **Lightweight:** Built with zero heavy external frameworks for fast loading times.

---

## 🛠️ Tech Stack

* **HTML5:** Structured semantic markup for the game board and card elements.
* **CSS3:** Custom styling, flexbox/grid layouts, and smooth 3D transform animations.
* **JavaScript (ES6+):** Game logic, deck shuffling algorithms, and state management.
