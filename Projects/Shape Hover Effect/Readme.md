# Interactive Shapes Hover Effect

A minimalist, interactive web project featuring geometric shapes that dynamically reveal vibrant background colors when hovered. Built entirely using clean HTML and CSS.

## 🚀 Features

* **Minimalist Design:** Clean geometric shapes centered on a soft pastel background.
* **Dynamic Hover States:** Moving your cursor over specific sections triggers a smooth color transition/appearance effect.
* **Pure CSS Layout:** Lightweight and fast, utilizing CSS flexbox/grid for positioning and CSS transitions for smooth animations.

## 🎨 Elements Included

1. **Shape 1:** Standard Square
2. **Shape 2:** Rotated Diamond
3. **Shape 3:** Perfect Circle

## 🛠️ Built With

* **HTML5:** For the structural markup of the shapes.
* **CSS3:** For styling, centering, positioning transforms, and the dynamic hover transitions.

---

### How to Run Locally

1. Clone this repository or download the source files.
2. Open the `index.html` file directly in any modern web browser.
3. Hover your mouse over the canvas and shapes to experience the color transitions!
